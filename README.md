# Cisco LAN Design Project

## Overview
This repository contains a comprehensive overview of the Cisco Packet Tracer LAN design project.

### Project Structure
This project features three distinct LANs, each designed to meet various network requirements:

1. **LAN 1: Corporate Network**  
   - This LAN includes essential components such as routers, switches, and desktop users to facilitate business operations.
   - It ensures robust connectivity and efficient data flow among various devices.

2. **LAN 2: Guest Network**  
   - This is a separate network designated for guest access, maintaining security and providing isolated access to the internet.
   - Wireless devices are integrated to allow flexibility for guest users.

3. **LAN 3: IoT Network**  
   - Specifically designed for Internet of Things (IoT) devices, this LAN manages connections and communications between smart devices and the wider network.

### Key Components
- **Wireless Devices:**  
  Wireless routers and access points are deployed across the LANs to provide mobile connectivity for laptops and handheld devices.

- **Routers:**  
  Several routers connect the LANs and manage traffic between them.

- **Switches:**  
  Each LAN is equipped with switches, which facilitate device connection within the network and improve performance.

- **Cloud Firewall:**  
  A cloud-based firewall secures the network by monitoring traffic and blocking unauthorized access.

- **IoT Devices:**  
  Various IoT devices, including smart sensors and automation devices, are integrated into the IoT network for a smart home or office environment.

## Conclusion
This project demonstrates the capabilities and functionalities of a modern LAN design using Cisco Packet Tracer, providing a solid foundation for future expansions and enhancements.
